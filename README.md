# Amazon Clone Website

A simple Amazon-inspired e-commerce homepage built using **HTML** and **CSS**. This project was created to practice frontend web development basics such as layouts, responsive design, navigation bars, product sections, and styling.

# Live Demo

Visit the website here:

**[https://aaamazonclone.netlify.app/](https://aaamazonclone.netlify.app/)**

---

# Features

* Amazon-style homepage layout
* Responsive navigation bar
* Product category sections
* Hero/banner section
* Footer section similar to Amazon
* Mobile responsive design improvements
* Clean UI using only HTML & CSS

---

# Technologies Used

* HTML5
* CSS3
